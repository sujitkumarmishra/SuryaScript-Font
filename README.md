WELCOME 
-------------------------------------------------------------------------------

Thank you for choosing the SuryaScript Font.

SuryaScript is a TrueType font developed to support the Surya Script Number
System. This font enables users to display and work with Surya Script
characters in compatible software applications while preserving the intended
appearance of the script.

-------------------------------------------------------------------------------
ABOUT SURYASCRIPT
-------------------------------------------------------------------------------

The Surya Script Number System is an independently developed writing system 
designed by Sujit Kumar Mishra. This font represents the digital implementation
of the script and is intended for educational, publishing, research, and
creative applications.

SuryaScript has been designed with a focus on readability, consistency, and
cross-application compatibility.

-------------------------------------------------------------------------------
INSTALLATION
-------------------------------------------------------------------------------

Windows

1. Close any applications currently using fonts.<br/>
2. Right-click "SuryaScript-Regular.ttf".<br/>
3. Select "Install" or "Install for all users".<br/>
4. Restart your applications if they were open during installation.<br/>
5. Other OS Supports macOS, Linux and Windows.<br/>

-------------------------------------------------------------------------------
USAGE
-------------------------------------------------------------------------------

1. Open your preferred application.<br/>
2. Select "Surya Script" from the font list.<br/>
3. Type using the supported keyboard layout or character mapping.<br/>
4. Save or print your document as needed.<br/>
